# Agent Notes

## Build & Run Commands

```bash
# Install dependencies
npm install

# Start development server (http://localhost:1234)
npm start

# Production build
npm run build
```

## Project Structure

- `src/Main.elm` - Main Elm application
- `src/index.js` - JavaScript entry with @smogon/calc
- `src/index.html` - HTML shell
- `src/styles.css` - Styling

## Elm Dependencies

Direct: elm/browser, elm/core, elm/html, elm/json, elm/url

## Key Ports

- `requestCalculation` / `receiveCalculation` - Damage calc
- `requestPokemonList` / `receivePokemonList` - Species data
- `requestMoveList` / `receiveMoveList` - Move data
- `requestItemList` / `receiveItemList` - Items
- `requestAbilityList` / `receiveAbilityList` - Abilities
- `requestNatureList` / `receiveNatureList` - Natures
